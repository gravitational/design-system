## Usage

Wrap the trigger element in `<Tooltip>` and pass the hint as the `content` prop.

```tsx
import { Button, Tooltip } from '@gravitational/design-system';

function Example() {
  return (
    <Tooltip content="Save your changes">
      <Button>Save</Button>
    </Tooltip>
  );
}
```

`content` accepts any `ReactNode`, so rich content (bold text, icons, multiple lines) works.

### Placements

Use the `placement` prop to position the tooltip relative to its trigger. Accepts any Chakra placement value (`top`, `top-start`, `top-end`, `right`, `bottom-start`, ...).

### Controlled

Drive the tooltip's visibility from parent state by passing `open` + `onOpenChange`.

```tsx
import { useState } from 'react';
import { Button, Tooltip } from '@gravitational/design-system';

function Example() {
  const [open, setOpen] = useState(false);
  return (
    <>
      <Button onClick={() => setOpen(o => !o)}>Toggle</Button>
      <Tooltip
        open={open}
        onOpenChange={details => setOpen(details.open)}
        content="Driven by parent state"
      >
        <Button>Hover target</Button>
      </Tooltip>
    </>
  );
}
```
