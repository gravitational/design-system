## Usage

Use `ComposedDialog` for the convenience API: pass a `trigger` and the body; Backdrop, Positioner, and Content are auto-injected. Wrap footer buttons in `Dialog.ActionTrigger` so they close the dialog automatically.

```tsx
import { Button, ComposedDialog, Dialog } from '@gravitational/design-system';

function Example() {
  return (
    <ComposedDialog trigger={<Button>Invite team members</Button>}>
      <Dialog.Header>
        <Dialog.Title>Invite team members</Dialog.Title>
      </Dialog.Header>
      <Dialog.Body>
        Add collaborators to this workspace. They'll receive an email with a
        link to join.
      </Dialog.Body>
      <Dialog.Footer>
        <Dialog.ActionTrigger asChild>
          <Button intent="neutral" fill="border">
            Cancel
          </Button>
        </Dialog.ActionTrigger>
        <Dialog.ActionTrigger asChild>
          <Button>Send invites</Button>
        </Dialog.ActionTrigger>
      </Dialog.Footer>
    </ComposedDialog>
  );
}
```

For a fully controlled dialog (no DOM trigger), omit `trigger` and drive it via `open` / `onOpenChange`. Backdrop + Positioner + Content are still auto-injected.

### Compose mode

For full compositional control over every slot, use `Dialog.Root` with the slot parts directly. This skips the auto-injection in `ComposedDialog`, letting you place the backdrop, positioner, and content wherever you want.

```tsx
import { Button, CloseButton, Dialog } from '@gravitational/design-system';

function Example() {
  return (
    <Dialog.Root>
      <Dialog.Trigger asChild>
        <Button>Update billing details</Button>
      </Dialog.Trigger>
      <Dialog.Backdrop />
      <Dialog.Positioner>
        <Dialog.Content>
          <Dialog.CloseTrigger asChild>
            <CloseButton size="sm" />
          </Dialog.CloseTrigger>
          <Dialog.Header>
            <Dialog.Title>Update billing details</Dialog.Title>
          </Dialog.Header>
          <Dialog.Body>Body content here.</Dialog.Body>
          <Dialog.Footer>
            <Dialog.ActionTrigger asChild>
              <Button intent="neutral" fill="border">
                Cancel
              </Button>
            </Dialog.ActionTrigger>
            <Dialog.ActionTrigger asChild>
              <Button>Save changes</Button>
            </Dialog.ActionTrigger>
          </Dialog.Footer>
        </Dialog.Content>
      </Dialog.Positioner>
    </Dialog.Root>
  );
}
```

### Alert dialog

For destructive or irreversible actions, set `role="alertdialog"`. Screen readers announce the dialog more assertively, and it's conventional to pair with a `danger` intent button and no close affordance in the header.

### With a form

Put form fields in `Dialog.Body`. Use `Dialog.ActionTrigger` on the submit button so a successful submission dismisses the dialog for free.

### Initial focus

By default the first focusable element receives focus on open. Pass `initialFocusEl` with a ref to target a specific element - usually the primary input for forms.

```tsx
import { useRef } from 'react';
import {
  Button,
  ComposedDialog,
  Dialog,
  Field,
  Input,
} from '@gravitational/design-system';

function Example() {
  const emailRef = useRef<HTMLInputElement>(null);
  return (
    <ComposedDialog
      initialFocusEl={() => emailRef.current}
      trigger={<Button>Invite a teammate</Button>}
    >
      <Dialog.Header>
        <Dialog.Title>Invite a teammate</Dialog.Title>
      </Dialog.Header>
      <Dialog.Body>
        <Field.Root>
          <Field.Label>Email</Field.Label>
          <Input ref={emailRef} type="email" />
        </Field.Root>
      </Dialog.Body>
    </ComposedDialog>
  );
}
```

### Open change

`onOpenChange` fires with the new open state for every dismissal (Escape, outside click, close button). Inspect `onEscapeKeyDown` and `onInteractOutside` if you need to distinguish the source.

### Controlled

Drive the dialog from parent state by passing `open` + `onOpenChange`. Omit the `trigger` prop; the dialog renders no DOM trigger of its own.

```tsx
import { useState } from 'react';
import { Button, ComposedDialog, Dialog } from '@gravitational/design-system';

function Example() {
  const [open, setOpen] = useState(false);
  return (
    <>
      <Button onClick={() => setOpen(true)}>Open</Button>
      <ComposedDialog
        open={open}
        onOpenChange={details => setOpen(details.open)}
      >
        <Dialog.Header>
          <Dialog.Title>Controlled dialog</Dialog.Title>
        </Dialog.Header>
        <Dialog.Body>Driven by parent state.</Dialog.Body>
        <Dialog.Footer>
          <Button intent="neutral" fill="border" onClick={() => setOpen(false)}>
            Cancel
          </Button>
          <Button onClick={() => setOpen(false)}>Save</Button>
        </Dialog.Footer>
      </ComposedDialog>
    </>
  );
}
```
