## Usage

Use `ComposedMenu` for the convenience API: pass a `trigger` and menu items as children.

```tsx
import { Button, ComposedMenu, Menu } from '@gravitational/design-system';

function Example() {
  return (
    <ComposedMenu placement="bottom-start" trigger={<Button>Actions</Button>}>
      <Menu.Item value="copy">Copy</Menu.Item>
      <Menu.Item value="paste">Paste</Menu.Item>
      <Menu.Item value="delete" color="fg.error">
        Delete
      </Menu.Item>
    </ComposedMenu>
  );
}
```
