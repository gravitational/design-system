## Usage

The `AbsoluteCenter` component is a wrapper around the `Box` component that applies styles to center its children both horizontally and vertically using absolute positioning.

The parent element must have a `position` of `relative`, `absolute`, or `fixed` for the centering to work correctly.

```tsx
import { AbsoluteCenter, Box } from '@gravitational/design-system';

<Box pos="relative" h="200px">
  <AbsoluteCenter>Centered Content</AbsoluteCenter>
</Box>;
```
