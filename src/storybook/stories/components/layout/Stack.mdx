## Usage

By default, `Stack` applies `flex-direction: column` and `gap: 8px` to its children.

```tsx
import { Stack } from '@gravitational/design-system';

function Example() {
  return (
    <Stack>
      <div>Item 1</div>
      <div>Item 2</div>
    </Stack>
  );
}
```
