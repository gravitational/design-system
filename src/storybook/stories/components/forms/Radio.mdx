## Usage

```tsx
import { ComposedRadioGroup } from '@gravitational/design-system';

function Example() {
  return (
    <ComposedRadioGroup
      label="Radio group"
      defaultValue="one"
      options={[
        { value: 'one', label: 'Option 1' },
        { value: 'two', label: 'Option 2' },
        { value: 'three', label: 'Option 3' },
      ]}
    />
  );
}
```

### Compose mode

For full compositional control over every slot, use `RadioGroup.Root` with the slot parts directly.

```tsx
import { RadioGroup } from '@gravitational/design-system';

function Example() {
  return (
    <RadioGroup.Root defaultValue="one">
      <RadioGroup.Label>Radio group</RadioGroup.Label>
      <RadioGroup.Item value="one">
        <RadioGroup.ItemHiddenInput />
        <RadioGroup.ItemIndicator />
        <RadioGroup.ItemText>Option 1</RadioGroup.ItemText>
      </RadioGroup.Item>
      <RadioGroup.Item value="two">
        <RadioGroup.ItemHiddenInput />
        <RadioGroup.ItemIndicator />
        <RadioGroup.ItemText>Option 2</RadioGroup.ItemText>
      </RadioGroup.Item>
    </RadioGroup.Root>
  );
}
```
